# What Muscles Do Pendlay Rows Work? Complete Anatomy Guide

The Pendlay row is a strict barbell row variation named after weightlifting coach Glenn Pendlay. Starting from a dead stop on the floor each rep, it eliminates momentum and forces explosive power from the back muscles. Here's the complete breakdown.

## Primary Muscles Worked

### Latissimus Dorsi

**Activation Level: Very High**

The lats are prime movers:
- Largest back muscle
- Shoulder extension
- Shoulder adduction
- Work explosively from dead stop
- Major back builder

### Rhomboids

**Activation Level: Very High**

Between the shoulder blades:
- Scapular retraction
- Maximum at top of row
- Upper back thickness
- Work hard each rep

### Middle and Lower Trapezius

**Activation Level: Very High**

The mid/lower trap fibers:
- Scapular retraction
- Scapular depression
- Important for back thickness

### Erector Spinae

**Activation Level: High**

The spinal erectors:
- Maintain spine position
- Isometric during pull
- Still significant work
- Less than bent-over rows (dead stop helps)

## Secondary Muscles Worked

### Rear Deltoids

**Activation Level: High**

Back of shoulders:
- Shoulder horizontal extension
- Pull with the lats
- Significant work

### Biceps Brachii

**Activation Level: High**

The biceps assist:
- Elbow flexion
- Part of pulling chain
- Work each rep

### Brachialis

**Activation Level: Moderate to High**

Under the biceps:
- Primary elbow flexor
- Assists the pull

### Core

**Activation Level: High**

Midsection braces:
- Protects the spine
- Stabilizes during explosive pull
- Works every rep

## What Makes Pendlay Rows Different

### Dead Stop Each Rep

Unlike bent-over rows:
- Bar rests on floor between reps
- Eliminates momentum and stretch reflex
- Each rep starts from zero
- Forces true concentric strength

### Explosive Intent

The movement is meant to be:
- Explosive pull from floor
- Fast to the chest
- Controlled return
- Power-focused

### Torso Position

More horizontal than typical rows:
- Back nearly parallel to floor
- More strict form required
- Greater lat involvement
- Less momentum possible

## Pendlay Row vs Bent-Over Row

| Factor | Pendlay Row | Bent-Over Row |
|--------|-------------|---------------|
| Starting position | Floor (dead stop) | Hanging |
| Momentum | None | Possible |
| Torso angle | More horizontal | Can vary |
| Explosive | Yes | Can be controlled |
| Continuous tension | No | Yes |
| Lower back demand | Less (dead stop rest) | More (holding position) |

**Pendlay:** Stricter, more explosive, dead stop
**Bent-over:** Continuous tension, more flexibility

## Muscle Activation by Phase

### Starting Position
- Bar on floor
- Bent over, back parallel
- Grip bar (overhand)
- Core braced, back flat

### Pull Phase (Concentric)
- Explosively pull to chest
- Lead with elbows
- Squeeze shoulder blades
- Fast, powerful movement

### Top Position
- Bar touches lower chest
- Scapulae fully retracted
- Brief moment
- Maximum back contraction

### Return Phase (Eccentric)
- Lower to floor with control
- Bar touches down
- Complete dead stop
- Reset for next rep

## Common Mistakes

**Not returning to floor**: Should be dead stop. Full rest on floor.

**Too much body movement**: Jerking upward. Stay horizontal.

**Rounding the back**: Dangerous. Flat back throughout.

**Pulling too high**: Should hit lower chest. Not neck.

**Slow, grinding reps**: Should be explosive. Pull fast.

**Bouncing off floor**: Not truly dead stop. Let it settle.

**Standing too upright**: Should be nearly horizontal. Stay bent over.

## How to Maximize Back Activation

1. **Dead stop each rep**: True rest on floor
2. **Explosive pull**: Fast, powerful movement
3. **Stay horizontal**: Back nearly parallel to floor
4. **Pull to lower chest**: Natural bar path
5. **Squeeze at top**: Retract scapulae hard
6. **Control down**: Don't just drop
7. **Reset between reps**: Brief pause at bottom
8. **Mind-muscle connection**: Feel back working

## Programming Recommendations

**For Back Strength:**
- 4-5 sets of 5 reps
- Heavy weight
- Explosive each rep
- Full rest between sets

**For Back Development:**
- 3-4 sets of 6-8 reps
- Moderate-heavy weight
- Focus on squeeze
- Dead stop each rep

**In a Back Workout:**
Can be primary row:
1. Pull-ups or pulldowns
2. **Pendlay rows** (main row)
3. Cable rows or accessories
4. Rear delts

**For Powerlifters/Strength Athletes:**
1. **Pendlay rows** (explosive strength)
2. Additional back work
3. Carries deadlift carryover

## Benefits for Athletes

Pendlay rows build:
- Explosive pulling power
- Concentric strength (no stretch reflex)
- Starting strength from dead stop
- Back tightness for deadlifts and cleans

Used by:
- Olympic lifters
- Powerlifters
- Athletes needing explosive power

## Grip Options

| Grip | Effect |
|------|--------|
| Overhand (pronated) | Standard, balanced |
| Underhand (supinated) | More bicep, lower lat |
| Wide | More upper back |
| Shoulder width | Balanced |

**Most common:** Overhand, shoulder width or slightly wider.

## Why Glenn Pendlay?

Glenn Pendlay was a renowned weightlifting coach who popularized this rowing style. His emphasis on:
- Strict form
- Dead stop reps
- Explosive power
- No cheating

Made this variation a staple in strength training.

## Weight Selection

Because of the explosive nature:
- Start lighter than bent-over rows
- Build up as technique improves
- Form before load
- Should be able to pull fast

If the bar isn't moving quickly, it's too heavy for this variation.

## Key Takeaways

- **Primary muscles**: Lats (very high), rhomboids (very high), mid-traps (very high), erectors (high)
- **Secondary**: Rear delts (high), biceps (high), core
- **Dead stop**: Bar rests on floor between each rep
- **Explosive**: Pull fast and powerfully
- **Horizontal torso**: Stay parallel to floor
- **Strict form**: No momentum or cheating possible
- **Named after Glenn Pendlay**: Legendary weightlifting coach
- **Great for strength athletes**: Builds starting strength

The Pendlay row is one of the most effective rowing variations for building explosive back strength. The dead stop eliminates cheating and forces true strength from the lats and upper back. Use proper form, pull explosively, and include Pendlay rows for serious back development.
