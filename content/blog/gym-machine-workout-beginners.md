Gym machines are the perfect starting point for beginners. They're safe, guide your movement, and let you build strength before progressing to free weights. Here's your complete guide to machine training.

## Why Start with Machines?

**Safety:** Fixed path reduces injury risk
**Learning curve:** Easier to use correctly
**Isolation:** Target specific muscles effectively
**Confidence:** Less intimidating than free weights
**No spotter needed:** Safe to train alone

## Common Gym Machines Explained

### Chest Machines

**Chest Press Machine**
- **What it works:** Chest, front shoulders, triceps
- **How to use:**
  1. Adjust seat so handles are at chest level
  2. Grip handles, press forward
  3. Extend arms without locking elbows
  4. Return with control
- **Sets/Reps:** 3 x 10-12

**Pec Deck (Chest Fly Machine)**
- **What it works:** Chest (isolation)
- **How to use:**
  1. Adjust seat so pads are at chest level
  2. Place forearms on pads
  3. Squeeze pads together in front
  4. Return with control
- **Sets/Reps:** 3 x 12-15

### Back Machines

**Lat Pulldown**
- **What it works:** Lats (back width), biceps
- **How to use:**
  1. Adjust thigh pad for stability
  2. Grip bar wider than shoulders
  3. Pull bar to upper chest
  4. Let arms extend fully (with control)
- **Sets/Reps:** 3 x 10-12

**Seated Row Machine**
- **What it works:** Middle back, lats, biceps
- **How to use:**
  1. Sit with chest against pad
  2. Grip handles
  3. Pull toward body, squeeze shoulder blades
  4. Return with control
- **Sets/Reps:** 3 x 10-12

**Assisted Pull-Up Machine**
- **What it works:** Lats, biceps
- **How to use:**
  1. Set counterweight (higher = easier)
  2. Kneel or stand on platform
  3. Grip handles, pull up
  4. Lower with control
- **Sets/Reps:** 3 x 8-10

### Shoulder Machines

**Shoulder Press Machine**
- **What it works:** Shoulders, triceps
- **How to use:**
  1. Adjust seat so handles are at shoulder level
  2. Grip handles
  3. Press overhead
  4. Lower with control
- **Sets/Reps:** 3 x 10-12

**Lateral Raise Machine**
- **What it works:** Side shoulders
- **How to use:**
  1. Adjust pads at elbow level
  2. Raise arms out to sides
  3. Stop at shoulder height
  4. Lower with control
- **Sets/Reps:** 3 x 12-15

### Leg Machines

**Leg Press**
- **What it works:** Quads, glutes, hamstrings
- **How to use:**
  1. Sit with back flat against pad
  2. Feet hip-width on platform
  3. Press platform away
  4. Lower until knees are at 90°
- **Sets/Reps:** 3 x 10-12

**Leg Extension**
- **What it works:** Quads (isolation)
- **How to use:**
  1. Adjust pad to rest on lower shins
  2. Extend legs fully
  3. Squeeze at top
  4. Lower with control
- **Sets/Reps:** 3 x 12-15

**Leg Curl (Lying or Seated)**
- **What it works:** Hamstrings (isolation)
- **How to use:**
  1. Position pad at ankles
  2. Curl weight toward glutes
  3. Squeeze at top
  4. Lower with control
- **Sets/Reps:** 3 x 10-12

**Hip Abductor Machine**
- **What it works:** Outer thighs, glutes
- **How to use:**
  1. Sit with pads against outer thighs
  2. Push legs apart
  3. Control the return
- **Sets/Reps:** 3 x 15

**Hip Adductor Machine**
- **What it works:** Inner thighs
- **How to use:**
  1. Sit with pads against inner thighs
  2. Squeeze legs together
  3. Control the return
- **Sets/Reps:** 3 x 15

**Calf Raise Machine**
- **What it works:** Calves
- **How to use:**
  1. Position shoulders under pads
  2. Rise onto toes
  3. Lower heels below platform level
  4. Rise fully and squeeze
- **Sets/Reps:** 3 x 15-20

### Arm Machines

**Bicep Curl Machine**
- **What it works:** Biceps
- **How to use:**
  1. Adjust seat so arms rest on pad
  2. Curl weight up
  3. Squeeze at top
  4. Lower with control
- **Sets/Reps:** 3 x 12

**Tricep Extension Machine**
- **What it works:** Triceps
- **How to use:**
  1. Adjust seat appropriately
  2. Push handles down
  3. Extend arms fully
  4. Return with control
- **Sets/Reps:** 3 x 12

### Core Machines

**Ab Crunch Machine**
- **What it works:** Abs
- **How to use:**
  1. Position pads on chest/shoulders
  2. Crunch forward
  3. Control the return
- **Sets/Reps:** 3 x 15

**Rotary Torso Machine**
- **What it works:** Obliques
- **How to use:**
  1. Sit with pad against chest
  2. Rotate to one side
  3. Control back to center
  4. Do both sides
- **Sets/Reps:** 3 x 12 each side

## How to Progress on Machines

### Week 1-4: Learn the Movements
- Light weight
- Focus on form
- Full range of motion
- 3 sets of 12-15 reps

### Week 5-8: Build Strength
- Increase weight slightly
- 3 sets of 10-12 reps
- Last rep should be challenging

### Week 9-12: Progressive Overload
- Add weight when you can complete all reps with good form
- Small increases (5-10 lbs)
- Track your weights

### Beyond Week 12
- Continue progressing on machines
- Start adding free weights (dumbbells first)
- Keep favorite machines in routine

## Common Beginner Mistakes

**Mistake 1: Too Much Weight**
Start light. Learn the movement before adding resistance.

**Mistake 2: Using Momentum**
Control every rep. No swinging or bouncing.

**Mistake 3: Partial Range of Motion**
Full range = full results. Extend and contract completely.

**Mistake 4: Skipping Leg Machines**
Legs are half your body. Don't avoid leg day.

**Mistake 5: No Progression**
Track weights. Aim to improve over time.

**Mistake 6: Rushing Between Machines**
Rest 60-90 seconds between sets for recovery.

## Machine Settings Guide

**Seat Height:** Most important adjustment
- Chest/shoulder machines: Handles at chest/shoulder level
- Leg machines: Knees at 90° when in starting position
- Back machines: Pad at chest level

**Weight Stack:** Start with the lightest weight that provides resistance

**Range of Motion:** Adjust stoppers to allow full ROM without discomfort

## Transitioning to Free Weights

After 8-12 weeks of machine training, start adding:

1. **Dumbbells:** Dumbbell press, rows, lunges
2. **Barbells:** Squat, bench press, deadlift
3. **Bodyweight:** Push-ups, pull-ups, dips

Keep machines for:
- Isolation work
- Safe failure training
- High-rep finishers

## Summary

Machines are excellent for beginners:

1. **Start with full body** 3x per week
2. **Learn each machine** with light weight
3. **Progress gradually** as you get stronger
4. **Track your weights** in a notebook or app
5. **Add free weights** after 8-12 weeks

Machines build the foundation of strength and movement patterns that transfer to free weights. Don't rush past them—they're valuable tools at any fitness level.
